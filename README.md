# SuperAuth

A centralized authentication system for web applications. Utilizes Supabase OAuth 2.1 Server for providing OAuth authentication facility.
Learn more about Supabase OAuth 2.1 Server [here](https://supabase.com/docs/guides/auth/oauth-server).

## Features

- OAuth authentication server
- Profile management
